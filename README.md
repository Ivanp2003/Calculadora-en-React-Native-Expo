# Bienvenido a la  app con Expo

Este es un proyecto creado con Expo usando `create-expo-app`.

---

##  Cómo empezar

### 1. Instalar dependencias
```bash
npm install
```
### 2. Iniciar la aplicación
```bash
npx expo start
```
### 3. Desarrollo

Puedes empezar a programar editando los archivos dentro de la carpeta:
```bash
app
```
Este proyecto usa un sistema de rutas basado en archivos (file-based routing).

### 4. Reiniciar el proyecto

Si quieres empezar desde cero, ejecuta:
```bash

npm run reset-project
```
Esto moverá el código actual a la carpeta:
```bash
app-example
```
y creará una nueva carpeta app vacía para que empieces desde cero.
